# PyTetris

This is a graphical Tetris game (tkinter).  
Just for Python3.

## Install tkinter

The version is 8.6 for tkinter.

```sh
$ git https://github.com/antoinealexisb/PyTetris.git
$ cd PyTetris

$ pip3 install -r requirements.txt

```

## Running Locally

```sh
$ git https://github.com/antoinealexisb/PyTetris.git
$ cd PyTetris

$ python3 pytetris.py
```
